### Building

#### Linux (with GCC 8.2 in minimum)

Building of this project need to pre-install CMake


```
sudo apt install cmake
cd $SRC_Dir
mkdir build
cd build
cmake ..
make 
```

### Run

```
cd $SRC_Dir/build
./Assignment2/ASSIGNMENT2_MAIN
```
